TTH Manager este o aplicație internă pentru Tales and Tech HUB.

Utilizatori:
- admin
- trainer

Scop:
- gestionare ateliere programate
- gestionare copii
- prezență
- observații
- traineri
- setări

Meniu:
- Dashboard
- Ateliere
- Copii
- Prezență
- Traineri
- Setări

Reguli:
- adminul poate adăuga și edita copii, ateliere și asignări
- trainerul vede toate atelierele
- trainerul poate marca prezența și observații doar la atelierele lui
- UI în română
- design modern, responsive, mobile-first
